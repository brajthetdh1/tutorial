Introduction to AWS

Amazon Web Services (AWS) is a cloud computing platform that provides a wide range of services including computing power, storage options, and databases. AWS is widely used in real-world applications and interviews.

In this guide, you will learn how to start AWS services and get prepared for interview questions.

AWS Setup & Launch Steps

1. EC2 (Elastic Compute Cloud)

  1. Log in to AWS Management Console.
  2. Go to EC2 Dashboard → Launch Instance.
  3. Choose Amazon Machine Image (AMI) e.g., Ubuntu or Amazon Linux.
  4. Select instance type (t2.micro for free tier).
  5. Configure instance details & add storage.
  6. Configure security group (allow SSH port 22).
  7. Launch instance and download key pair (.pem).
  8. Connect via terminal: ssh -i "key.pem" ec2-user@public-ip

2. S3 (Simple Storage Service)

  1. Go to S3 console → Create bucket.
  2. Give a unique bucket name and select region.
  3. Set permissions and access (public/private as needed).
  4. Upload files and organize folders.
  5. Enable static website hosting if needed.

3. Lambda (Serverless)

  1. Go to Lambda console → Create function.
  2. Select "Author from scratch" and choose runtime (Python, Node.js, etc.).
  3. Configure permissions & trigger (e.g., API Gateway, S3).
  4. Write function code or upload zip.
  5. Test function using test events or API trigger.

4. RDS (Relational Database Service)

  1. Go to RDS console → Create database.
  2. Select engine (MySQL, PostgreSQL, etc.) and version.
  3. Choose free tier template if available.
  4. Set DB instance identifier, username & password.
  5. Configure network & security group.
  6. Launch database and connect using endpoint.

AWS Interview Questions

  • What is AWS and why is it used?
  • Explain the difference between EC2 and Lambda.
  • What is the difference between S3 and EBS?
  • How do security groups differ from NACLs?
  • Explain vertical and horizontal scaling in AWS.
  • What is IAM and why is it important?
  • How do you connect an EC2 instance to an RDS database?
  • What is AWS Free Tier?
  • How do you deploy a static website on S3?
  • What are CloudWatch metrics and alarms?
